# Interactive-Bipedal-robot
This project runs bujii.py on a Raspberry Pi to control an 8-servo bipedal robot through voice commands, with speech recognition, text-to-speech, YouTube playback, and Gemini generative AI integration.

Required Electronics
Raspberry Pi 4 Model B (recommended)
8 x hobby servo motors (3.3V/5V compatible)
5V power supply for the Raspberry Pi and servos
Recommended: 5V 3A or higher
If servos draw a lot of current, use a separate 5V servo power supply
USB microphone or compatible Pi microphone
Speaker or headphones
Use Pi 3.5mm audio jack, USB audio, or HDMI audio
Breadboard / servo driver board / jumper wires
Common ground connection between Raspberry Pi and servo power supply
